# Changelog

Todas as mudanças importantes do projeto serão documentadas neste arquivo.

## [0.1.0] - 2025-11-03

### Added
- Estrutura inicial do projeto
- Backend FastAPI com endpoint de predição
- Frontend Angular básico
- Scripts ETL para Google Forms
- Pipeline de treinamento Random Forest
- Análise SHAP para explicabilidade
- Templates dos formulários Google Forms
- Configuração Docker Compose
- Schema PostgreSQL
- Documentação técnica
- Scripts de setup automatizado
- Makefile para automação

### Features
- Coleta de dados via Google Forms
- Processamento ETL automatizado
- Modelo Random Forest para predição de turnover
- API REST para predições em tempo real
- Dashboard web responsivo
- Sugestões personalizadas de retenção
- Compliance LGPD/GDPR

### Technical
- Python 3.11 com FastAPI
- Angular 17 para frontend
- PostgreSQL como banco de dados
- Scikit-learn para machine learning
- SHAP para explicabilidade
- Docker para containerização
